# Instacart Basket Analysis

Data analysis project using Python to explore customer purchasing behavior for Instacart.  
Completed as part of the **CareerFoundry Data Analytics Program**.

---

## Project Overview

The goal of this project is to analyze data from an online grocery retailer, **Instacart**, to uncover insights about customer purchasing behavior.  
Using Python and data visualization techniques, this analysis provides recommendations to improve marketing strategies and customer segmentation.

---

## Repository Contents

| Folder | Description |
|---------|-------------|
| **01 Project Management/** | Project brief and documentation outlining objectives and business questions. |
| **03 Scripts/** | Python scripts for cleaning, analyzing, and visualizing the Instacart data. |
| **04 Analysis/** | Includes summary reports, visualizations, and analytical findings. |
| **05 Sent to Client/** | Final presentation materials shared with stakeholders. |

---

## Key Analytical Questions

1. What are the busiest days and hours of the week for Instacart orders?  
2. When do customers spend the most money?  
3. Which customer segments (e.g., loyalty status, family size) exhibit specific purchasing patterns?  
4. How can Instacart tailor marketing strategies for different customer profiles?
